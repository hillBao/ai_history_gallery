> 一个基于 Vue 3 + Vite 的人工智能发展历史展示项目，以可视化和多媒体形式呈现AI从萌芽到现状的发展历程。

## 📦 快速开始

### 环境要求
- Node.js >= 16.0.0
- npm >= 8.0.0

### 安装与运行

```bash
# 进入项目目录
cd ai-history-gallery-vue

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

开发服务器默认运行在 http://localhost:5173

### 构建部署

```bash
# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

## 📋 项目需求文档

### 🎯 背景说明

随着人工智能（AI）技术的不断演进，各行业对AI发展脉络与里程碑事件的关注度持续攀升。为帮助访客快速了解AI技术的起源、重要节点和未来趋势，需设计并开发一个"AI发展历史页面"，以可视化和多媒体形式呈现AI从萌芽到现状的发展历程。

### 🎯 目标与范围

#### 总体目标
- 📊 通过时间轴、图文并茂的方式呈现AI发展过程中的关键事件与代表成果
- 🔗 提供丰富的外部链接与参考资料，便于用户深度阅读
- 🔧 保持页面的易用性与可维护性，能够随时更新新的里程碑

#### 功能范围
| 功能模块 | 描述 | 优先级 |
|---------|------|--------|
| 静态内容模块 | 包括AI发展简介、历史时间轴、关键人物与机构介绍 | 🔴 高 |
| 交互式时间轴组件 | 支持用户滚动或点击查看不同年份/阶段的详情 | 🔴 高 |
| 响应式设计 | 兼容 PC 端与移动端浏览 | 🟡 中 |
| 多语言支持 | 初步以中文为主，预留英文版本切换空间 | 🟢 低 |

## 🏗️ 页面结构与内容规划

### 页面框架概览

```
┌─────────────────────────────────────┐
│          顶部导航 (Header)            │
│  Logo + 导航菜单 (主页/AI概述/历史等)   │
├─────────────────────────────────────┤
│         主视觉区域 (Banner)           │
│     背景图片/视频 + 页面标题            │
├─────────────────────────────────────┤
│              内容区域                │
│  ┌─ 模块一：AI简介                   │
│  ├─ 模块二：时间轴总览               │
│  ├─ 模块三：详细里程碑               │
│  ├─ 模块四：重要人物与机构           │
│  ├─ 模块五：未来展望                │
│  └─ 模块六：参考资料与延伸阅读        │
├─────────────────────────────────────┤
│            底部 (Footer)            │
│      版权信息 + 社交媒体链接          │
└─────────────────────────────────────┘
```

### 📑 详细内容模块

#### 🔸 模块一：AI简介
- AI定义与起源简介
- 核心概念解释

#### 🔸 模块二：时间轴总览
| 时代 | 关键特征 | 代表事件 |
|------|---------|----------|
| 20世纪50-60年代 | AI概念萌芽 | 图灵测试、达特茅斯会议 |
| 70-80年代 | 专家系统兴起 | 知识工程、符号主义 |
| 90-00年代 | 机器学习发展 | 统计学习、神经网络复兴 |
| 10-20年代 | 深度学习突破 | AlphaGo、GPT系列 |

#### 🔸 模块三：详细里程碑
每个里程碑包含：
- 📅 年份
- 📝 事件名称
- 📖 事件简介
- 🖼️ 相关图片/视频
- 🔗 相关链接

#### 🔸 模块四：重要人物与机构
- **学术先驱**：图灵、麦卡锡、赫布等
- **知名机构**：Google DeepMind、OpenAI等
- **代表成果**：重要论文、产品等

#### 🔸 模块五：未来展望
- 🔮 技术趋势预测
- 🚀 发展方向解读
- ⚠️ 挑战与机遇

#### 🔸 模块六：参考资料与延伸阅读
- 📚 权威书籍推荐
- 📄 重要论文集
- 🌐 官方网站链接
- 📰 新闻报道精选

## 🎨 UI/UX 设计要点

### 🎨 配色与风格
- **主色调**：蓝色或紫色系，体现科技感
- **设计风格**：简洁的扁平化风格，减少过度装饰
- **交互效果**：时间线节点使用卡片式展示，支持微交互动效

### 📖 排版与可读性
- **字体选择**：无衬线字体（Source Sans Pro、Roboto）
- **字体大小**：正文不小于16px
- **行间距**：1.5倍行高
- **段落间距**：不低于24px
- **模块分割**：明确分割，留白适当

### 🖱️ 互动体验

#### 时间轴交互
- **PC端**：鼠标拖拽或滚轮滚动
- **移动端**：手势滑动切换
- **节点点击**：展开详情面板

#### 导航功能
- **返回顶部**：固定在右下角的快速按钮
- **锚点导航**：滚动时对应模块高亮显示
- **进度指示**：页面滚动进度条

### 📊 可视化元素
- **时间轴图表**：重大里程碑的可视化展示
- **统计图表**：各阶段研究热点比例（可选）
- **人物肖像**：重要人物的头像展示
- **事件插图**：关键事件的示意图

## 🚀 技术栈

- **前端框架**：Vue 3
- **构建工具**：Vite
- **样式方案**：CSS3 + 预处理器
- **图表库**：Chart.js / D3.js（可选）
- **动画库**：GSAP / Lottie（可选）
## 📁 组件结构

项目使用 Vue 3 的单文件组件，每个主要模块对应一个组件：

- `Header` 顶部导航
- `Banner` 主视觉区域
- `modules/*` 功能模块组件
- `Footer` 底部信息
